# 斬 - Chanbara Online 改

ブラウザで遊べる1対1の3Dチャンバラゲーム **完全版**。Three.js で描画し、PeerJS (WebRTC P2P) で友達と直接対戦できます。サーバー不要、GitHub Pages だけで動作します。

## 主な特徴

- **人型ヒューマノイドキャラ**: 頭・目・眉・口・髷・首・上半身/下半身・関節のある腕脚・足袋まで再現
- **マウスで刀を振り回せる**: 左ボタン押しながらマウスを動かすと刀がスイング、勢いが乗れば自動で斬撃が発動
- **致命傷・部位ダメージ**: 頭(1.8倍) / 胴 / 脚(0.75倍) の部位判定、会心(クリティカル)演出付き
- **スタミナ & 気合ゲージ**: 攻撃/回避でスタミナ消費、被弾/与ダメで気合が溜まり **必殺技 (Q)** が発動
- **アイテム 4種**: 回復薬・煙玉・手裏剣・気合丹（**1〜4** で持ち替え、**E** で使用）
- **マップ 6種**: 道場・桜庭・雪山・城下町・竹林・夕陽の浜（それぞれ天候・地形・移動補正あり）
- **刀 7種**: 打刀・太刀・脇差・野太刀・小烏丸・村雨(高クリ率)・草薙(必殺で雷)
- **AI 対戦 / オンライン対戦 (PeerJS) / 練習(素振り)**
- **キーボードで完結する UI**: 矢印キー＋Enter で刀・マップを選択可能（決定ボタン押せない問題は完全解消）
- **サウンド** (Web Audio 合成)、カメラシェイク、致命傷ビネット、桜吹雪・雪・火の粉などの天候演出

## 遊び方

1. ページを開いて名前を入力
2. モード選択
   - **AI と 立ち合う** — CPU と対戦
   - **部屋を 開く / 入る** — 友達と P2P オンライン対戦
   - **素振り (練習)** — 動かない案山子で操作確認
3. **刀を選ぶ** → **マップを選ぶ** → 出陣
4. 先に **2 本** 取った方の勝ち

## 操作 (片手 1 本指で完結 ── Ragdoll Blade 風)

| 動作 | やり方 |
|------|--------|
| **歩く** | 画面に指を置いたまま、ゆっくり 指を動かす。起点からの ズレ た方向に歩く |
| **斬る** | 同じ指を 素早く 払うように動かす（指の速度がそのまま 刀の勢いに）|
| **強斬り** | より素早く・大きく 払う（自動で強斬りに切替）|
| **刀を振り回す** | 指でぐるぐる 円を描く と 刀も回る (溜め演出)|
| **必殺技** | 右下の **必** ボタン（気合ゲージ満タンで発動）|
| **受け (ガード)** | 右下の **受** ボタンを押している間 |
| **回避** | 右下の **避** ボタン (無敵フレーム) |
| **アイテム** | 右下の **薬** ボタン or アイテムバーをタップ |
| **メニューへ** | 右上の **×** ボタン |

PC で遊ぶ場合はマウス左ボタンを押しながら動かすことで 同じ操作 が可能。

### 爽快感を支える演出

- **ヒットストップ** ── 敵を斬った瞬間、画面が一瞬止まる
- **振動** ── スマホが ビリッ と短く震える (会心は二段の波形で強め)
- **カメラシェイク** ── 当たった瞬間に揺れる
- **刀の残光** ── 速く振るほど 軌跡が伸びる

```bash
# http://localhost:8000 を開く
```
