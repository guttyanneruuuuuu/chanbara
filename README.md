# 斬 - Chanbara Online

ブラウザで遊べる1対1の3Dチャンバラゲーム。Three.jsで描画し、PeerJS(WebRTC P2P)で友達と直接対戦できます。サーバー不要、GitHub Pagesだけで動作します。

## 遊び方

1. ページを開く
2. 名前を入れる
3. モードを選ぶ
   - **AI と 立ち合う** — CPU と対戦
   - **部屋を 開く** — 合言葉を生成して友達を待つ（ホスト）
   - **部屋に 入る** — 合言葉を入れて参加
   - **素振り (練習)** — 動かない案山子で操作確認
4. 刀を選ぶ
5. 試合開始（先に 2 本取った方の勝ち）

### 操作

| キー | 動作 |
|------|------|
| WASD | 移動 |
| マウス | 視点（クリックでポインタロック） |
| 左クリック | 斬撃 |
| Shift + 左クリック | 強斬り（威力大・隙大） |
| 右クリック（押している間） | 受け（ガード） |
| Space | 回避（無敵フレームあり） |
| ESC | メニューに戻る |

## 刀の種類

| 名前 | 特徴 |
|------|------|
| 打刀 | バランス型 |
| 太刀 | 間合いが広いが少し重い |
| 脇差 | 短く軽快、手数で押す |
| 野太刀 | 大太刀、一撃必殺だが鈍重 |
| 小烏丸 | 銘刀、鋭く軽妙 |

## オンライン対戦の仕組み

- **PeerJS**(WebRTCシグナリング)を利用したP2P接続
- ホストが生成する**合言葉(5文字)**を友達に伝えるだけで接続可能
- 中継サーバーを自前で用意する必要なし
- 静的サイト(GitHub Pages)だけで完結

> ※ 一部の厳しいNAT/企業ネットワークではP2P接続できない場合があります。

## ローカルで動かす

```bash
python3 -m http.server 8000
# http://localhost:8000 を開く
```

## ファイル構成

```
index.html   # UI / メニュー / HUD
style.css    # 和風シックなデザイン

favicon.svg  # アイコン
```

## 技術スタック

- [PeerJS](https://peerjs.com/) — WebRTC P2P 通信
